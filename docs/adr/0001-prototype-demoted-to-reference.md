# Prototype demoted from visual spec of record to reference

Through Phase 7, `prototype/VenueDash_Prototype.dc.html` was the visual spec of record ("port screens, don't redesign"). Decided 2026-07-07, at the start of the v0.5 UI/flow pass: the prototype is now **reference material only**. It was drawn for the v1.0 vision before v0.5 trimmed scope, so several of its screens describe flows that no longer exist (held deposits, e-sign, claims), and the built app has diverged for good reasons — the thing to optimize is the actual v0.5 workflow, not fidelity to a pre-scope drawing. The prototype files themselves remain untouched and useful as a visual vocabulary reference. Note a rename/brand refresh may follow v0.5, which would supersede the prototype's visual language entirely.
